# Airline Data Challenge - PySpark
**Solution using PySpark**

---

## 📋 Problem Statement

This project analyzes **airline data** to help a new airline enter the US domestic market.  
The goal is to recommend **5 profitable round-trip routes** between medium and large US airports.

### Key Requirements:
1. Top 10 busiest round-trip routes
2. Top 10 most profitable round-trip routes
3. Recommend 5 best routes for investment
4. Calculate breakeven flights for $90M airplane cost
5. Suggest future KPIs to track
